Como vamos rastrear o Trilho do Membro?

O membro avança por várias etapas: Novo Membro -> Consolidação -> Encontro -> Escola. Como devemos salvar isso?

A

Apenas Status Atual (Simples)

Adicionamos um campo `etapaAtual` no perfil do membro. É rápido de implementar, mas não guarda o histórico (não saberemos a data que ele fez o Encontro com Deus no passado).


Membro {
  nome: "João",
  etapaAtual: "PRE_ENCONTRO"
}
B

Histórico de Conclusões (Recomendado)

Criamos uma tabela de histórico. Assim, registramos exatamente em qual data o membro concluiu cada etapa do trilho. Essencial para estatísticas da igreja.


Membro { nome: "João" }

HistoricoEtapas [
  { etapa: "NOVO_MEMBRO", data: "10/01/2026" },
  { etapa: "CONSOLIDACAO", data: "15/02/2026" }
]
